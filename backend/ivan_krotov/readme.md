## **Leiko**
```
Miinused: Paginationit pole. parameeter "name" ei tööta , peab oleam "org_name". Koodisisene dokumenteerimine puudub. Tehtud readme on väga pealiskaudne ja ei anna vajalikku infot et lahendus tööle saada. Kui andmebaasis vasteid pole siis enamus päringute puhul viskab koodi erroreid.
Plussid: Kood ise on väga põhjalikult üles ehitatud. Päris selliseid üleliikseid või mõttetuid koodiosasi ei täheldanud. Põhimõtteliselt toimib.
Arvamus: Arvestades oskuste pagasit siis ootaks paremat tulemust. Vähemasti oleks võinud ülesandes nõutud osa tehtud olla.
```
